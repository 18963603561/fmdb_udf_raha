# 标注模板长相示意图

本文用于说明采样后导出的 Excel 标注模板大致长相，以及用户应该如何填写标注结果。

## 整体结构

导出的文件是 `.xls` 格式，包含四个工作表：

| 工作表 | 是否用户填写 | 用途 |
|---|---|---|
| 标注数据 | 是 | 展示采样行和业务字段，用户在这里填写标注结果 |
| 标注说明 | 否 | 说明 `_row_label`、`_error_columns`、`_comment` 的填写规则 |
| 导入校验 | 否 | 导入失败或部分失败时，系统回写错误行和错误原因 |
| 系统信息 | 否 | 保存模板版本、数据集、采样批次、字段列表等系统元数据，通常隐藏 |

## 标注数据工作表

用户主要看到的是“标注数据”工作表。系统字段通常被隐藏或保护，业务字段只用于查看，不建议修改。

| _annotation_task_id | _row_id | _row_content_hash | _display_row_no | name | age | address | _row_label | _error_columns | _comment |
|---|---|---|---:|---|---:|---|---:|---|---|
| task-001 | row-001 | hash-001 | 1 | 张三 | 18 | 北京市朝阳区 | 0 |  | 正常 |
| task-002 | row-002 | hash-002 | 2 | 李四 | 999 | 上海市浦东新区 | 1 | age | 年龄明显异常 |
| task-003 | row-003 | hash-003 | 3 | 王五 | 25 |  | 1 | address | 地址为空 |
| task-004 | row-004 | hash-004 | 4 | 赵六 | 130 |  | 1 | age,address | 年龄异常且地址为空 |

## 字段含义

| 字段 | 来源 | 是否用户填写 | 说明 |
|---|---|---|---|
| `_annotation_task_id` | 系统生成 | 否 | 标注任务标识，用于回灌时校验这一行属于原采样任务 |
| `_row_id` | 系统生成 | 否 | 采样逻辑行标识，用于定位原始采样行 |
| `_row_content_hash` | 系统生成 | 否 | 导出时的行内容哈希，用于防止业务字段被误改后继续导入 |
| `_display_row_no` | 系统生成 | 否 | 展示序号，便于人工阅读 |
| 业务字段 | 采样数据 | 否 | 例如 `name`、`age`、`address`，用于辅助判断这一行是否异常 |
| `_row_label` | 用户填写 | 是 | `0` 表示整行正常，`1` 表示整行异常 |
| `_error_columns` | 用户填写 | 是 | 异常行填写错误字段名，多个字段使用英文逗号分隔 |
| `_comment` | 用户填写 | 否 | 可选备注，说明判断原因 |

## 用户填写规则

| 场景 | `_row_label` | `_error_columns` | 示例 |
|---|---:|---|---|
| 整行正常 | 0 | 留空 | `0`、空 |
| 单个字段异常 | 1 | 填异常字段名 | `1`、`age` |
| 多个字段异常 | 1 | 英文逗号分隔 | `1`、`age,address` |
| 只有备注说明 | 0 或 1 | 按实际情况填写 | `_comment` 写判断理由 |

注意：`_error_columns` 必须使用模板里的业务字段名，不能写中文描述，也不能写不存在的字段名。

## 标注说明工作表

| 字段 | 说明 |
|---|---|
| `_row_label` | `0` 表示正常，`1` 表示异常 |
| `_error_columns` | 异常行填写异常字段名，多个字段使用英文逗号分隔 |
| `_comment` | 可选标注说明 |

## 导入校验工作表

首次导出时这个工作表只有表头。导入失败或部分失败时，系统会生成带错误信息的工作簿。

| Excel行号 | _row_id | 错误编码 | 错误说明 |
|---:|---|---|---|
| 3 | row-002 | COLUMN_NOT_FOUND | 异常字段名不在模板字段列表中 |
| 5 | row-004 | ROW_CONTENT_MISMATCH | 行内容哈希与可信采样快照不一致 |

## 系统信息工作表

该工作表保存导入校验所需的元数据，用户不需要填写。

| key | value |
|---|---|
| templateVersion | xls-v1 |
| datasetId | 示例数据集标识 |
| samplePartitionMonth | 示例采样月分区 |
| sampleBatchId | 示例采样批次 |
| schemaHash | 示例字段结构哈希 |
| businessColumnsJson | `["name","age","address"]` |
| detectableColumnsJson | `["name","age","address"]` |
| recordCount | 4 |
| exportedAt | 示例导出时间戳 |

## 导入后的含义

导入服务会把每一行的整行标注展开成单元格标签：

| 用户填写 | 系统理解 |
|---|---|
| `_row_label=0` | 这一行参与审核的字段都被视为正常标签 |
| `_row_label=1` 且 `_error_columns=age` | `age` 字段是异常标签，其他参与审核字段是正常标签 |
| `_row_label=1` 且 `_error_columns=age,address` | `age` 和 `address` 是异常标签，其他参与审核字段是正常标签 |
